Vim Text Editor Commands

"Vim Text Editor Commands in Linux and Git Bash"

Open Vim

vim FileName.txt

Exit Vim

press :q to quit

press :wq to write/save then quit

press :w to write/save only

press :q! to don't save then quit

Vim Modes

Vim has two modes : Command Mode and Insert mode


Command mode: used for commands like Quit , Write , Write and Quit and so on ...


Insert Mode: used for text Editing


When you open Vim it's by default in Command mode


Vim Command Mode

in Command mode you can move around in the text using:


  • j: Move Down
  • k: Move Up
  • l: Move Right
  • h: Move Left

Vim Insert Mode

To switch to Insert Mode Press i


in Insert Mode you can write/Edit the file

To get back to Command mode Press ESC

Vim Delete/Undelete Line Command

In Command Mode : Press dd to delete a line


In Command Mode : Press u to undelete/undo a line


Vim Redo/Undo Line Command

In Command Mode : Press u to undo an action


In Command Mode : Press CTRL+R to Redo an action


Vim Go Top/Bottom Command

In Command Mode : Press shift+g to to go to the bottom of the file


In Command Mode : Press gg to to go to the top of the file


Vim Skip a block Command

In Command Mode : Press { (one time or hold the press to skip more) to to go to the bottom of a block of lines


In Command Mode : Press } (one time or hold the press to skip more) to to go to the top of the a block of lines


Vim Skip Number of Lines

You can skip Number of lines: Press number of lines then hit j for down and k for up or l for right and h for left
